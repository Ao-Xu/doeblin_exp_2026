# Doeblin-Anchored Contrastive Learning for Markov Transition Kernels

Reproduction code for the paper "Doeblin-Anchored Contrastive Learning for Markov Transition Kernels".

## Overview

This repository contains the complete experimental pipeline for validating the theoretical framework of Doeblin-anchored contrastive learning. The experiments cover:

1. **End-to-end calibration** (Exp 1): Validates the calibration chain from contrastive risk to density error
2. **Markovization** (Exp 2): Tests the validity restoration operator
3. **Rates & dimension** (Exp 3): Empirical convergence rates across dimensions
4. **Anchor strength & reference law** (Exp 4): Tradeoff analysis for anchor parameter ε
5. **Trajectory learning** (Exp 5): Beta-mixing trajectory experiments with thinning
6. **Dynamical transfer** (Exp 6): Finite-horizon rollout and stationary distribution error
7. **Ablation study** (Exp 7): Component-wise ablation of the pipeline
8. **Runtime scalability** (Exp 8): Computational cost analysis

## Requirements

```
Python >= 3.7
numpy >= 1.19
matplotlib >= 3.3
```

Install dependencies:
```bash
pip install -r requirements.txt
```

## Quick Start

Run all experiments (v1 + v2):
```bash
cd experiments
python run_experiments.py      # v1 experiments → results/
python run_experiments_v2.py   # v2 experiments → results_v2/
```

## Key Components

### `run_experiments_v2.py` (v2)
Extended experiments with neural contrastive training:
- `ReLUContrastiveNet`: Clipped ReLU sieve for anchored contrastive learning
- `train_contrastive_net`: Training loop with validation selection
- 8 experiment functions covering calibration, Markovization, rates, anchor, trajectory, dynamics, ablation, runtime

## Reproducibility Notes

- All experiments use fixed random seeds (seed=42 by default)
- Neural networks are trained with simple gradient descent (no GPU required for these synthetic experiments)
- Runtime experiments report wall-clock times on a single CPU core
- Results are deterministic given the same numpy/matplotlib versions
